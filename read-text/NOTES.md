# Read Text - OCR with Azure AI Vision

## Overview
Application that extracts and annotates text from images using Azure Computer Vision's OCR (Optical Character Recognition) capabilities.

## Key Features
- **Text Extraction**: Reads text from images using Azure AI Vision READ API
- **Line Detection**: Identifies complete lines of text with bounding polygons
- **Word Recognition**: Extracts individual words with confidence scores
- **Visual Annotation**: Creates annotated images showing detected text regions
- **Multiple Image Support**: Processes various image formats (JPG, PNG, etc.)
