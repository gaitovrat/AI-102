# DALL-E Client - Notes

## Overview
Interactive application that generates images from text prompts using Azure OpenAI's DALL-E 3 model.

## How It Works
1. User enters text prompt describing desired image
2. Sends prompt to DALL-E 3 API for image generation
3. Receives image URL from API response
4. Downloads and saves image as PNG file
5. Continues loop until user types 'quit'

## Use Case
Demonstrates Azure OpenAI's generative AI capabilities for creating custom images from natural language descriptions.
