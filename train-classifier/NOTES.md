# Train Classifier - Azure Custom Vision Training

## Overview
Trains a custom image classification model using Azure Custom Vision by uploading tagged images and initiating training iterations.

## Key Features
- **Batch Image Upload**: Uploads images organized in folders by tag/class
- **Automated Tagging**: Uses folder names as classification tags
- **Model Training**: Trains Custom Vision model and monitors progress
- **Multi-class Classification**: Supports multiple categories (e.g., fruits)
