# Knowledge Mining - Search Application

## Overview
Python application that demonstrates Azure AI Search (formerly Cognitive Search) for knowledge mining and document search with enriched metadata.

## Key Features
- **Full-text search** across indexed documents stored in Azure Blob Storage
- **Enriched metadata extraction**: locations, people, and key phrases from documents
- **Interactive CLI**: Query loop for multiple searches in a session
- **Sorted results**: Documents ordered by filename
