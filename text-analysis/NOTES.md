# Text Analysis Notes

## Overview
Azure AI Language Service demo for analyzing text reviews with NLP capabilities.

## Features Demonstrated
1. **Language Detection** - Identifies primary language of text
2. **Sentiment Analysis** - Determines positive/negative/neutral sentiment
3. **Key Phrase Extraction** - Extracts important phrases from text
4. **Entity Recognition** - Identifies named entities (people, places, organizations)
5. **Entity Linking** - Links entities to knowledge bases with URLs
