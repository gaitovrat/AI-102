# QnA App - Analysis Notes

## Purpose
Q&A application using Azure AI Language Service's Question Answering feature for knowledge base queries.

## Workflow
1. Load environment variables
2. Initialize QuestionAnsweringClient with credentials
3. Enter question loop
4. Submit questions to knowledge base
5. Display answers with confidence scores and source attribution

## Use Case
Enterprise knowledge base queries, FAQ automation, customer support, documentation search.
